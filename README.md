## Backend
(Do it one time)
  python -m venv myenv
	myenv\Scripts\activate
	pip install django
	cd backend
	pip install djangorestframework
	pip install django-cors-headers
	python -m pip install Pillow
	python manage.py migrate
 
(To run the project)
  npm start
  python manage.py runserver
